# Интерактивный тест «Анафилактический шок»

Компактный статический сайт для итогового тестирования среднего медицинского персонала ГБУЗ «ГП №134 ДЗМ». Работает на телефоне и компьютере без базы данных и установки.

## Что внутри

- 12 вопросов, по одному на экран;
- мгновенная проверка и краткое пояснение;
- прогресс, итоговые баллы, процент и оценка;
- поля ФИО и подразделение;
- повторное прохождение;
- доступная клавиатурная навигация и адаптивная вёрстка.

ФИО и результат отображаются только на устройстве пользователя и никуда не отправляются.

## Публикация на GitHub Pages

1. Откройте созданный репозиторий на GitHub.
2. Загрузите в корень репозитория файлы `index.html`, `style.css` и `script.js` (можно также загрузить этот `README.md`).
3. Откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**.
5. Выберите ветку **main**, папку **/(root)** и нажмите **Save**.
6. Подождите несколько минут. На этой же странице появится адрес опубликованного теста.

Обычно адрес имеет вид:

`https://ИМЯ-ПОЛЬЗОВАТЕЛЯ.github.io/ИМЯ-РЕПОЗИТОРИЯ/`

## QR-код

После публикации скопируйте адрес теста. QR-код можно создать прямо в браузере Chrome: откройте страницу теста → меню браузера → **Сохранить и поделиться → Создать QR-код**. Скачанный PNG можно вставить в презентацию.

## Важно

Это учебный материал, а не самостоятельный клинический протокол. Перед использованием ответственному медицинскому работнику следует сверить формулировки с действующими клиническими рекомендациями и локальными алгоритмами учреждения.
